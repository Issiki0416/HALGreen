# マルチログイン
### それぞれモデル、マイグレーションを作成し、RouteServiceProviderをいじる
- ユーザー(商品を購入)・・・/
- オーナー(商品を登録)・・・/owner
- 管理者(オーナーの管理)・・・/admin
    - オーナー新規登録
        - フラッシュメッセージで登録完了を表示する


- 提供側
    - 商品の登録
    - 商品の編集
- 利用側
    - 商品購入
